Non SQL / Not Only SQL
- Non-relational DataBase([[NRDB]])를 다루기 위한 프로그래밍 언어
# 특징 ([[SQL]]과의 차이점)
## Scheme가 없다
- SQL은 row, column과 같이 schema를 정하고, 이를 따르는 데이터만 추가할 수 있다.
- NoSQL은 schema를 따르지 않는 데이터도 추가할 수 있다.
